## 🏗️ Things I've Built (and Survived)

### 🎯 **VisionGuard — Real-time Threat Detection System**
A computer vision pipeline that:
- Detects anomalous behavior in video streams using YOLO + custom classifiers
- Runs inference at the edge with optimized TensorRT models
- Pushes alerts via a lightweight backend with sub-100ms latency

**Stack:** OpenCV, PyTorch, TensorRT, FastAPI, Docker

---

### ☁️ **CloudPilot — Multi-cloud Deployment Orchestrator**
An internal DevOps tool that:
- Abstracts away AWS/GCP differences behind a single YAML config
- Auto-provisions infra with Terraform + spins up K8s clusters
- Monitors cost drift and sends budget alerts in real-time

**Stack:** Terraform, Kubernetes, Python, AWS, GCP, Grafana

---

### 🧠 **NeuralDeploy — MLOps Pipeline Framework**
A framework for ML engineers who hate YAML hell:
- Packages models → containerizes → deploys to cloud endpoints
- Tracks experiments and model versions automatically
- One command from notebook to production

**Stack:** MLflow, Docker, FastAPI, Python, CI/CD
